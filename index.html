Lesson 2: Creating a Structured Document with HTML

Browser Developer Tools

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them

HTML Tree Structure

The "tree-like structure" comes from the fact that HTML elements can have other elements inside of them.
In a browser, this structure shows up as a series of nested boxes

HTML - Indentation & Boxes

When you read an HTML document as text, you see changing indentations going up and down the left side of the document. The more indented an element is, the more deeply nested it's corresponding "box" is.

Text Editors

When writing code, you can use special text editors (like Sublime Text for example).

Lesson 3: Adding CSS Style to HTML Structure

Avoiding Repitition Repitition

Avoiding repitition in your code is very important.
Repitition means additional work, replicating the same code over and over manually, which can be very time-consuming.
Additionally, it can lead to errors if you fail to update the code manually each time it appears.
Avoiding repitition helps with code clarity & consistency.

What is CSS?

CSS stands for cascading style sheets.
The style "cascades" across a document, and is inherited through all of the elements in the top element.

Including CSS on a Webpage

1. Write CSS into head of HTML - good for small projects
2. Link HTML file to separate CSS file - better for larger projects. Put a link to the CSS file in the head of your HTML doc & make sure that file is in the same folder as the HTML.
3. Write style inline with HTML - involves lots of repetition. generally a bad idea.

Boxes within Boxes within Boxes

The Box Model
Margin - space between boxes
Border - liner around the padding
Padding - space inside the box that protects the content
Content - nucleus

Boxes can be set as a fixed number of pixels or as a percentage of the screen.

The Flex Box Layout provides an efficient way to lay out items inside a container or div.
